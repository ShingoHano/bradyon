bradyon
======================

It is a tab browser application for iOS which was characterized  
by two screen display function, auto-reload function, the touch pad operation.

Features
------

1. 5-Tab-Browser
2. Two Screen Display
3. Auto-Reload
4. Touch Pad Operation
5. Google Suggest
